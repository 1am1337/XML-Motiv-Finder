# MXL Motiv Finder
A simple search for a (currently hard coded) musical motiv in the music21 corpus and chosen MXL Files

Reqiurements:
- music21
- pandas
- matplotlib

Usage:
Currently just executing the fileSearcher.py file.

To edit the searched-for motiv change the "BachMotivNotes" variable. has to be in the proper music21.note.Note format

To edit the searched corpus change the bachCorpus variable. also has to adhere to all the music21 standards

Searched MXL-Files are in the KdF/ folder.
